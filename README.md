# vietlott-suggest-app
Just another random number project for 6/45 lotto.

Result and suggestions given by this code is for Vietnamese only, but this application is not a Vietlott's application.

At the time of coding this application, Vietlott has no exposed API for 3rd party application retrieving updated data. All data is manually updated, so if it's out of date, please not be annoyed :D
